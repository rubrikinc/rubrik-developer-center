# isOrgServiceAccountDisabled

Returns whether the service accounts of the organization are not enabled.

## Returns

Boolean!

## Sample

=== "Query"

    ```graphql
    query {
      isOrgServiceAccountDisabled
    }
    ```

=== "Variables"

    ```json
    {}
    ```

=== "Example Response"

    ```json
    {
      "data": {
        "isOrgServiceAccountDisabled": true
      }
    }
    ```
