# ThreatMonitoringStats

Aggregation stats for threat monitoring in the selected time range.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| objectsWithMatches | [Long](../scalars/Long.md)! | Number of objects scanned with threat matches found. |
| objectsWithNoMatches | [Long](../scalars/Long.md)! | Number of objects scanned with no threat matches found. |
| totalObjectsScanned | [Long](../scalars/Long.md)! | Total number of objects scanned. |
