# AuthorizedPrincipalEdge

Wrapper around the AuthorizedPrincipal object. This wrapper is used for pagination.

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [AuthorizedPrincipal](AuthorizedPrincipal.md)! | The actual AuthorizedPrincipal object wrapped by this edge. |
