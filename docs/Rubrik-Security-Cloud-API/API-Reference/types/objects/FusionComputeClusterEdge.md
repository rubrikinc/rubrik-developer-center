# FusionComputeClusterEdge

Wrapper around the FusionComputeCluster object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [FusionComputeCluster](FusionComputeCluster.md)! | The actual FusionComputeCluster object wrapped by this edge. |
