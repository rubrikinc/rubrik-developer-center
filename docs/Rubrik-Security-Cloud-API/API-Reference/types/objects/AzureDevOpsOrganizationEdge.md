# AzureDevOpsOrganizationEdge

Wrapper around the AzureDevOpsOrganization object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [AzureDevOpsOrganization](AzureDevOpsOrganization.md)! | The actual AzureDevOpsOrganization object wrapped by this edge. |
