# CassandraColumnFamilyEdge

Wrapper around the CassandraColumnFamily object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [CassandraColumnFamily](CassandraColumnFamily.md)! | The actual CassandraColumnFamily object wrapped by this edge. |
