# HierarchySnappableEdge

Wrapper around the HierarchySnappable object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [HierarchySnappable](../interfaces/HierarchySnappable.md)! | The actual HierarchySnappable object wrapped by this edge. |
