# GithubOrganizationEdge

Wrapper around the GithubOrganization object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [GithubOrganization](GithubOrganization.md)! | The actual GithubOrganization object wrapped by this edge. |
