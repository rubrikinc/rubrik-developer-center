# AzureManagedDiskMetadata

AzureManagedDiskMetadata represents the metadata for the Azure Managed Disk.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| attachedVmId | String! | The ID of the Azure virtual machine to which the managed disk is attached. |
| attachedVmName | String! | The name of the Azure virtual machine to which the managed disk is attached. |
