# O365SharepointObjectEdge

Wrapper around the O365SharepointObject object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [O365SharepointObject](../interfaces/O365SharepointObject.md)! | The actual O365SharepointObject object wrapped by this edge. |
