# AzureNativeResourceGroupEdge

Wrapper around the AzureNativeResourceGroup object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [AzureNativeResourceGroup](AzureNativeResourceGroup.md)! | The actual AzureNativeResourceGroup object wrapped by this edge. |
