# KubernetesClusterEdge

Wrapper around the KubernetesCluster object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [KubernetesCluster](KubernetesCluster.md)! | The actual KubernetesCluster object wrapped by this edge. |
