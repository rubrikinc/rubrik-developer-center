# KubernetesVirtualMachineEdge

Wrapper around the KubernetesVirtualMachine object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [KubernetesVirtualMachine](KubernetesVirtualMachine.md)! | The actual KubernetesVirtualMachine object wrapped by this edge. |
