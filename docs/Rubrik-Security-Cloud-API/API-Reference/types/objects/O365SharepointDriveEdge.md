# O365SharepointDriveEdge

Wrapper around the O365SharepointDrive object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [O365SharepointDrive](O365SharePointDrive.md)! | The actual O365SharepointDrive object wrapped by this edge. |
