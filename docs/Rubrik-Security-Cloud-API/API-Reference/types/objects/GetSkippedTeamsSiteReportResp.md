# GetSkippedTeamsSiteReportResp

Response for the skipped Teams site report for Sharepoint bulk recovery.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| externalDownloadId | String! | A report of the workloads restored. |
| totalSkippedSiteCount | Int! | It is the total number of skipped sites. |
