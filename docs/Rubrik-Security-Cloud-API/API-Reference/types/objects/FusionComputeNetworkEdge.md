# FusionComputeNetworkEdge

Wrapper around the FusionComputeNetwork object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [FusionComputeNetwork](FusionComputeNetwork.md)! | The actual FusionComputeNetwork object wrapped by this edge. |
