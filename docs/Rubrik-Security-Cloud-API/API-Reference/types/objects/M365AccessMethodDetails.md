# M365AccessMethodDetails

Details of how the principal can access an M365 item.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| itemName | String! | Name of the M365 item where the access method is defined. |
| itemPath | String! | Path of the M365 item where the access method is defined. |
| sharingLink | String! | Sharing link for the access method. |
