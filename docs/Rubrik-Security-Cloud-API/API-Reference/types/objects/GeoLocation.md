# GeoLocation

Geographic location information for a Rubrik cluster, including physical address and coordinates.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| address | String! | Physical address or location name of the Rubrik cluster. |
| latitude | Float! | Latitude coordinate of the cluster location in decimal degrees. |
| longitude | Float! | Longitude coordinate of the cluster location in decimal degrees. |
