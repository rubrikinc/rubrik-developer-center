# FilterGroupConfig

FilterGroupConfig represents a group of filters with a logical operator.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| filtersList | [[PolicyFilter](PolicyFilter.md)!]! | A list of filters in this group. |
| logicalOperator | [LogicalOperator](../enums/LogicalOperator.md)! | The logical operator applied to this group of filters. |
