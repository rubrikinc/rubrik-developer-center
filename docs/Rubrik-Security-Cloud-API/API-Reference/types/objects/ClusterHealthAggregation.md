Aggregate Rubrik clusters' health information.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| fatal | [Long](../scalars/Long.md)! | The number of Rubrik clusters that have a FATAL status. |
| ok | [Long](../scalars/Long.md)! | The number of Rubrik clusters that have an OK status. |
| warning | [Long](../scalars/Long.md)! | The number of Rubrik clusters that have a WARNING status. |
