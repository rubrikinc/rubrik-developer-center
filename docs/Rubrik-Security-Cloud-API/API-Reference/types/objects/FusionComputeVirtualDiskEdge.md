# FusionComputeVirtualDiskEdge

Wrapper around the FusionComputeVirtualDisk object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [FusionComputeVirtualDisk](FusionComputeVirtualDisk.md)! | The actual FusionComputeVirtualDisk object wrapped by this edge. |
