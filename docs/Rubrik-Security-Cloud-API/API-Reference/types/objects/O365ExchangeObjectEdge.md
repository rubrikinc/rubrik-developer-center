# O365ExchangeObjectEdge

Wrapper around the O365ExchangeObject object. This wrapper is used for pagination.

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [O365ExchangeObject](../interfaces/O365ExchangeObject.md)! | The actual O365ExchangeObject object wrapped by this edge. |
