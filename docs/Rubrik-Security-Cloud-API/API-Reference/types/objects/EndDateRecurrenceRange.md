# EndDateRecurrenceRange

A recurrence range with a start and end (e.g. repeat the pattern from 7/29/2019 until 4/13/2022).

## Fields

| Field | Type | Description |
|-------|------|-------------|
| endDate | [DateTime](../scalars/DateTime.md)! | The ending date of the recurrence. |
| startDate | [DateTime](../scalars/DateTime.md)! | The starting date of the recurrence. |
