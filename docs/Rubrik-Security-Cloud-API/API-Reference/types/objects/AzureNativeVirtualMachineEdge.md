# AzureNativeVirtualMachineEdge

Wrapper around the AzureNativeVirtualMachine object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [AzureNativeVirtualMachine](AzureNativeVirtualMachine.md)! | The actual AzureNativeVirtualMachine object wrapped by this edge. |
