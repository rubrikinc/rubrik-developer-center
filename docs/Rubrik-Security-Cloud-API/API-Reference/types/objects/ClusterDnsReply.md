# ClusterDnsReply

Represents the response of a request to retrieve DNS information from a Rubrik cluster.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| domains | [String!]! | List of DNS search domains. |
| servers | [String!]! | List of DNS name servers. |
