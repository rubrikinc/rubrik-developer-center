# AzureSqlDatabaseServerEdge

Wrapper around the AzureSqlDatabaseServer object. This wrapper is used for pagination.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cursor | String! | String used to identify this edge. |
| node | [AzureSqlDatabaseServer](AzureSqlDatabaseServer.md)! | The actual AzureSqlDatabaseServer object wrapped by this edge. |
