# BulkOnDemandSnapshotNutanixVmInput

Input to initiate Bulk on Demand snapshot for Nutanix.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| config | [NutanixBulkOnDemandSnapshotJobConfigInput](NutanixBulkOnDemandSnapshotJobConfigInput.md)! | Required. Configuration objects. Each object includes an identifier for the virtual machine and the configuration of the virtual machine backup. |
| userNote | String | User note to associate with audits. |
