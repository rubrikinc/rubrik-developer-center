# SharePointListItem

Represents the SharePoint list item to be restored.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| itemId | String! | ID of the item to restore. |
| itemName | String! | Name of the item to restore. |
| itemSnapshotsToRestore | [[SharePointListItemSnapshot](SharePointListItemSnapshot.md)!]! | List of snapshots of this item to restore. |
