# MapAzureCloudAccountExocomputeSubscriptionInput

Input for mapping Azure cloud accounts to an Exocompute subscription.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| cloudAccountIds | [[UUID](../scalars/UUID.md)!]! | Rubrik IDs of the cloud accounts to be mapped. |
| exocomputeCloudAccountId | [UUID](../scalars/UUID.md)! | Rubrik ID of the Exocompute cloud account to map to for launching Exocompute. |
