# O365FullSpExclusion

Configuration for the exclusion of a SharePoint object from protection.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| fid | String! | The fid of the SharePoint object. |
| name | String! | The name of the SharePoint object. |
| objectType | [SharePointDescendantType](../enums/SharePointDescendantType.md)! | The object type. |
| url | [URL](../scalars/URL.md)! | The URL of the SharePoint object. |
