# FusionComputeVmRequestStatusInput

Input for querying the status of an asynchronous FusionCompute request.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| clusterUuid | [UUID](../scalars/UUID.md)! | Required. UUID used to identify the cluster the request goes to. |
| id | [UUID](../scalars/UUID.md)! | Required. ID of an asynchronous request. |
