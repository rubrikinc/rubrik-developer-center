# MongoConfigInput

Input to configure the SLA Domain for MongoDB database.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| logFrequency | [SlaDurationInput](SlaDurationInput.md) | Frequency value for log backup of MongoDB databases. |
| logRetention | [SlaDurationInput](SlaDurationInput.md) | Specifies the duration for which the MongoDB database logs will be retained. |
