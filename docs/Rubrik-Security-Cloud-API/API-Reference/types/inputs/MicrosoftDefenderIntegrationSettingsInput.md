# MicrosoftDefenderIntegrationSettingsInput

Holds the settings for a Microsoft Defender integration.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| minSeverity | [DefenderAlertSeverity](../enums/DefenderAlertSeverity.md) | Minimum alert severity to ingest. Alerts at this severity and above will be included. UNSPECIFIED means default (LOW). |
