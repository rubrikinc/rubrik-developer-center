# VcenterPreAddConfigInput

Supported in v6.0+

## Fields

| Field | Type | Description |
|-------|------|-------------|
| connectionConfig | [VcenterConnectionConfigInput](VcenterConnectionConfigInput.md) | Supported in v6.0+ IP address and account credentials of the vCenter server that is being added. Specify this if the vCenter is not added to Rubrik cluster. |
| id | String | Supported in v6.0+ Id of the vCenter. Specify this if the vCenter is already added to the Rubrik cluster. |
