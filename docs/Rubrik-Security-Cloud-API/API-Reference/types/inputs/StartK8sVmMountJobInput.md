# StartK8sVmMountJobInput

Input for starting a Kubernetes virtual machine mount job.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| config | [K8sVmMountParametersInput](K8sVmMountParametersInput.md)! | Required. Configuration for the mount job. |
| id | [UUID](../scalars/UUID.md)! | Required. The ID of the virtual machine snapshot to be mounted. |
