# AwsNativeS3SlaConfigInput

AWS Native S3 SLA configuration.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| archivalLocationId | String | Specifies the location ID where the primary backups will be stored. |
| continuousBackupRetentionInDays | Int | Specifies the number of days for which the AWS S3 continuous backups will be retained. |
