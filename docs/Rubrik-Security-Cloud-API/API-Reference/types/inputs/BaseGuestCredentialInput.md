# BaseGuestCredentialInput

Supported in v5.0+

## Fields

| Field | Type | Description |
|-------|------|-------------|
| password | String! | Required. Supported in v5.0+ Password for the account used to login to the VM guest OS. |
| username | String! | Required. Supported in v5.0+ Username for the account used to login to the VM guest OS. To include a domain, use the format <domain>\<username>. |
