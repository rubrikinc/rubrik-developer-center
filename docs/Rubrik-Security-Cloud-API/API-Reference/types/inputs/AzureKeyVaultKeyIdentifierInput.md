| Field | Type | Description |
|-------|------|-------------|
| keyName | String! | Azure Key Vault key name. |
| keyVersion | String! | Azure Key Vault key version. |
| kmsKeyVaultId | [UUID](../scalars/UUID.md)! | Azure Key Vault ID. |
