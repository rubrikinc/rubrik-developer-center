# ClusterUuidWithMssqlObjectIdInput

Contains the cluster UUID and a Microsoft SQL object ID.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| clusterUuid | [UUID](../scalars/UUID.md)! | Specifies the UUID used to identify the cluster that the request goes to. |
| mssqlObjectId | String | ID of a Microsoft SQL object. |
