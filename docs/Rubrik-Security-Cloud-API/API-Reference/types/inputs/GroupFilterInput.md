# GroupFilterInput

Input for filtering user groups.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| authDomainIdsFilter | [String!] | Filter according to the authentication domain ID. |
| nameFilter | String | Filter user groups by name. |
| orgIdsFilter | [String!] | Filter user groups by organization IDs. |
| roleIdsFilter | [String!] | Filter user groups that have the specified roles assigned in the current organization. |
