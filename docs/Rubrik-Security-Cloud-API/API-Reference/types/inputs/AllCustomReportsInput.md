# AllCustomReportsInput

Retrieves all custom reports with filtering options.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| createdBy | String | Optional filter by creator user ID. |
| reportCategory | [ReportCategory](../enums/ReportCategory.md) | Optional filter by report category. |
| reportRoom | [ReportRoomType](../enums/ReportRoomType.md) | Optional filter by report room. |
| reportViewType | [PolarisReportViewType](../enums/PolarisReportViewType.md) | Optional filter by report view type. |
| searchTerm | String | Optional search term to filter by report name. |
