# DayOfWeekPatternInput

Day-of-week pattern specification. For example, First Monday, Last Friday.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| dayOfWeek | [DayOfWeek](../enums/DayOfWeek.md) | Specifies the day of the week. For example, MONDAY, TUESDAY, SUNDAY. |
| weekOrdinal | [WeekOrdinal](../enums/WeekOrdinal.md) | Specifies the ordinal of the week in that month. For example, FIRST, SECOND, THIRD, FOURTH, LAST. |
