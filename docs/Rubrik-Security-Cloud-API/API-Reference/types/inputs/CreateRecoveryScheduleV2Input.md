# CreateRecoveryScheduleV2Input

Input for creating a recovery schedule for the specified recovery plan.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| recoveryPlanId | [UUID](../scalars/UUID.md)! | Recovery plan identifier for which this schedule will be created. |
| scheduleInfo | [ScheduleInfoV2](ScheduleInfoV2.md)! | Schedule information. |
