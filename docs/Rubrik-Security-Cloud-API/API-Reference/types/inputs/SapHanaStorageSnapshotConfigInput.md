# SapHanaStorageSnapshotConfigInput

Input to configure the SLA Domain for SAP HANA Storage Snapshot.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| frequency | [SlaDurationInput](SlaDurationInput.md) | Frequency value for storage snapshot of SAP HANA systems. |
| retention | [SlaDurationInput](SlaDurationInput.md) | Number of days for which storage snapshot of SAP HANA systems will be retained. |
