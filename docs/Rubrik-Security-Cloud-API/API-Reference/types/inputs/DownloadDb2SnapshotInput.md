# DownloadDb2SnapshotInput

Input for downloading a Db2 snapshot from an archival location.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| locationId | String! | Required. ID of the location from where the Db2 database snapshot will be downloaded. |
| snapshotId | String! | Required. ID of the Db2 database snapshot. |
| userNote | String | User note to associate with audits. |
