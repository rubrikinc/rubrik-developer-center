# AllReportTemplatesByCategoriesInput

Retrieves all report templates by categories.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| reportCategory | [ReportCategory](../enums/ReportCategory.md) | The category of the report templates to retrieve. |
| reportRoom | [ReportRoomType](../enums/ReportRoomType.md) | Optional filter by report room. |
| searchTerm | String | Optional search term to filter the report templates. |
