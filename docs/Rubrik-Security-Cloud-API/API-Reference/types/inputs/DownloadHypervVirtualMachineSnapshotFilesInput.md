# DownloadHypervVirtualMachineSnapshotFilesInput

Input for downloading Hyper-V snapshot files.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| config | [HypervDownloadFilesJobConfigInput](HypervDownloadFilesJobConfigInput.md)! | Configuration information for a job to download files and folders from a Hyper-V virtual machine backup. |
| id | String! | ID assigned to a Hyper-V virtual machine backup object. |
| userNote | String | Required. User note to associate with audits. |
