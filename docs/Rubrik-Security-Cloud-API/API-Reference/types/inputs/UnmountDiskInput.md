# UnmountDiskInput

Input required to unmount disks.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| liveMountId | [Long](../scalars/Long.md)! | Live Mount ID for which disk must be unmounted. |
| mountIds | [[Long](../scalars/Long.md)!]! | List of Mount IDs to be unmounted. |
| targetWorkloadId | [UUID](../scalars/UUID.md)! | Target Workload ID for which disk must be unmounted. |
