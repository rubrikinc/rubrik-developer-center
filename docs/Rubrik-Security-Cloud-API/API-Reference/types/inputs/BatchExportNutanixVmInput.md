# BatchExportNutanixVmInput

Input to export a batch Nutanix virtual machines.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| config | [NutanixBatchExportSnapshotJobConfigInput](NutanixBatchExportSnapshotJobConfigInput.md)! | Required. Configuration objects. Each object includes an identifier for the virtual machine, an identifier for the snapshot of the virtual machine, and the configuration of the exported virtual machine snapshot. |
