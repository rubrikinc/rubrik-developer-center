# SnapshotTimeFilter

SnapshotTimeFilter specifies the time range for snapshot queries. Exactly one of before_time or after_time must be set.

## Fields

| Field | Type | Description |
|-------|------|-------------|
| afterTime | [DateTime](../scalars/DateTime.md) | Point in time to search after. |
| beforeTime | [DateTime](../scalars/DateTime.md) | Point in time to search before. |
