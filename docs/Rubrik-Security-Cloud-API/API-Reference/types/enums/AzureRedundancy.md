# AzureRedundancy

Redundancy value for the Azure resource. For more information, see https://docs.microsoft.com/en-us/azure/storage/common/storage-redundancy.

## Values

| Value | Description |
|-------|-------------|
| GRS | Geo-redundant storage. |
| GZRS | Geo-zone-redundant storage. |
| LRS | Locally redundant storage. |
| RA_GRS | Read access Geo-redundant storage. |
| RA_GZRS | Read access Geo-zone-redundant storage. |
| UNKNOWN_AZURE_REDUNDANCY | Unknown Azure storage redundancy. |
| ZRS | Zone-redundant storage. |
