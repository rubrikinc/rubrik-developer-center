# EventClusterType

Enum representing all the cluster types.

## Values

| Value | Description |
|-------|-------------|
| CLOUD | Cloud Cluster. |
| EXO_COMPUTE | Exocompute cluster. |
| ON_PREM | On-premises cluster. |
| ROBO | ROBO cluster. |
| RUBRIK_SAAS | Rubrik SaaS cluster. |
| UNKNOWN_CLUSTER_TYPE | Unknown cluster type. |
