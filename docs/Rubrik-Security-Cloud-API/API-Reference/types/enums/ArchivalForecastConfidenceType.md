# ArchivalForecastConfidenceType

Confidence level of the archival storage forecast.

## Values

| Value | Description |
|-------|-------------|
| FORECAST_CONFIDENCE_HIGH | High confidence: sufficient upload history and calibration data. |
| FORECAST_CONFIDENCE_LOW | Low confidence: insufficient historical data for accurate forecasting. |
| FORECAST_CONFIDENCE_MEDIUM | Medium confidence: limited data or recent SLA changes. |
