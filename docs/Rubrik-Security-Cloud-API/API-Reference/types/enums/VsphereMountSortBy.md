# VsphereMountSortBy

Sort by field for vSphere mount results.

## Values

| Value | Description |
|-------|-------------|
| CLUSTER_NAME | Sort by cluster name. |
| MOUNT_TIME | Sort by mount time. |
| NEW_VM_NAME | Sort by new VM name. |
| SOURCE_VM_NAME | Sort by source VM name. |
| STATUS | Sort by status. |
