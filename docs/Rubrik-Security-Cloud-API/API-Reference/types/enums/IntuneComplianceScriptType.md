# IntuneComplianceScriptType

Specifies the script type of an Intune compliance script.

## Values

| Value | Description |
|-------|-------------|
| INTUNE_COMPLIANCE_SCRIPT_TYPE_POWERSHELL | The script type is PowerShell script. |
| INTUNE_COMPLIANCE_SCRIPT_TYPE_SHELL | The script type is Shell script. |
| INTUNE_COMPLIANCE_SCRIPT_TYPE_UNKNOWN | The script type is unknown. |
