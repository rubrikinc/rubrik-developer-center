# LegalHoldMode

LegalHoldMode specifies how snapshot copies are retained under legal hold.

## Values

| Value | Description |
|-------|-------------|
| ALL_LOCATIONS | Retain snapshot copies at all locations (local, archival, replication). |
| ANY_LOCATION | Retain snapshot at one optimal location chosen by priority. |
| LEGAL_HOLD_MODE_UNSPECIFIED | Default unspecified legal hold mode. |
