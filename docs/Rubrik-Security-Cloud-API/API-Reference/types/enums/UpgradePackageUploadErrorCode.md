# UpgradePackageUploadErrorCode

The error code for the upload.

## Values

| Value | Description |
|-------|-------------|
| INTERNAL_FAILURE | Internal failure error. |
| MD5_MISMATCH | MD5 mismatch error. |
| NETWORK_FAILURE | Network failure error. |
| NO_ERROR | No error. |
| STORAGE_FULL | Storage full error. |
