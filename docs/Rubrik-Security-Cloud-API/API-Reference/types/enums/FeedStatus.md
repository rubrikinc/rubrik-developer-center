# FeedStatus

Status of the feed.

## Values

| Value | Description |
|-------|-------------|
| ACTIVE | Feed is active. |
| IMPORTING_IOCS | Feed is importing IOCs. |
| INVALID_CREDENTIALS | Feed credentials are invalid. |
| NEW_DISABLED_IOCS_INGESTED | Feed ingested deactivated IOCs. |
| UNSPECIFIED | Status is UNSPECIFIED. |
