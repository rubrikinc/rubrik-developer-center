# SalesforceObjectBackupType

SalesforceObjectBackupType indicates whether a Salesforce object is recommended for backup or not.

## Values

| Value | Description |
|-------|-------------|
| NOT_RECOMMENDED | Object is not recommended for backup. |
| RECOMMENDED | Object is recommended for backup. |
| SALESFORCE_OBJECT_BACKUP_TYPE_UNSPECIFIED | Unspecified backup type. |
