# AzureSqlLtrRetentionUnit

AzureSqlLtrRetentionUnit defines the time unit for long-term retention (LTR) periods in Azure SQL databases.

## Values

| Value | Description |
|-------|-------------|
| DAYS | Retention period in days. |
| MONTHS | Retention period in months. |
| WEEKS | Retention period in weeks. |
| YEARS | Retention period in years. |
