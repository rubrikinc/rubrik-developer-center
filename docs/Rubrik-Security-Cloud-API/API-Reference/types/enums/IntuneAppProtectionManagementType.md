# IntuneAppProtectionManagementType

Specifies the management type of an Intune app protection policy.

## Values

| Value | Description |
|-------|-------------|
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_CONFIG_MANAGER_CLIENT | Configuration Manager client management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_CONFIG_MANAGER_CLIENT_MDM | Configuration Manager client and MDM management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_CONFIG_MANAGER_CLIENT_MDM_EAS | Configuration Manager client, MDM, and EAS management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_EAS | Exchange ActiveSync management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_EAS_INTUNE_CLIENT | EAS and Intune client management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_EAS_MDM | EAS and MDM management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_GOOGLE_CLOUD_DEVICE_POLICY_CONTROLLER | Google Cloud Device Policy Controller management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_INTUNE_CLIENT | Intune client management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_MDM | Mobile device management type. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_UNKNOWN | The management type is unknown. |
| INTUNE_APP_PROTECTION_MANAGEMENT_TYPE_WITHOUT_ENROLLMENT | Management type without enrollment. |
