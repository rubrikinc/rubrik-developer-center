# CassandraSourceStatus

Enum for cassandra source status.

## Values

| Value | Description |
|-------|-------------|
| ADDING | Source is getting added. |
| CONNECTED | Source is connected. |
| DELETED | Source is deleted. |
| DELETING | Source is getting deleted. |
| DISCONNECTED | Source is disconnected. |
| REFRESHING | Source data is getting refreshed. |
| UNKNOWN_SYSTEM_STATUS | Unknown source status. |
