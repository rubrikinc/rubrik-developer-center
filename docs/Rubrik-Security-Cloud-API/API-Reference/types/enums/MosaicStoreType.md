# MosaicStoreType

Type of storage location.

## Values

| Value | Description |
|-------|-------------|
| AZURE_STORE | Microsoft Azure Store. |
| GS_STORE | Google Cloud Store. |
| NFS_STORE | NFS Store. |
| S3_STORE | AWS S3 Store. |
| TYPE_UNAVAILABLE | Store Type is not available for this store. |
