# QmcInitiatorPage

Page from which a QMC quarantine operation was initiated.

## Values

| Value | Description |
|-------|-------------|
| QMC_FILES | QMC files page. |
| QMC_OBJECTS | QMC objects page. |
| QMC_OBJECT_DETAILS | QMC object details page. |
| QMC_UNSPECIFIED | Unspecified initiator page. |
