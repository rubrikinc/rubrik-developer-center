# HypervMountedVmStatusType

The connection status of Hyper-V Host server.

## Values

| Value | Description |
|-------|-------------|
| MOUNTED | Hyper-V Live Mount is Mounted. |
| MOUNTING | Hyper-V Live Mount is Mounting. |
| POWEREDOFF | Hyper-V Live Mount is Powered Off. |
| POWEREDON | Hyper-V Live Mount is Powered On. |
| POWERINGOFF | Hyper-V Live Mount is Powering Off. |
| POWERINGON | Hyper-V Live Mount is Powering On. |
| UNKNOWN | Hyper-V Live Mount virtual machine status could not be parsed. |
| UNMOUNTING | Hyper-V Live Mount is Unmounting. |
