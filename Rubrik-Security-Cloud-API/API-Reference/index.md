# RSC GraphQL API Reference

The RSC GraphQL API has a single endpoint: `POST /api/graphql`. Use the sidebar to browse [queries](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/queries/index.md), [mutations](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/mutations/index.md), and [types](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/index.md), or use search to find a specific operation.

- [Changelog](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/Changelog/index.md) — schema changes by version
- [Deprecations](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/Deprecations/index.md) — fields and types deprecated in the current schema
