# MicrosoftSiteEdge

## Fields

| Field  | Type                                                                                                                           | Description                                           |
| ------ | ------------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------- |
| cursor | String!                                                                                                                        | String used to identify this edge.                    |
| node   | [MicrosoftSite](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/interfaces/MicrosoftSite/index.md)! | The actual MicrosoftSite object wrapped by this edge. |
