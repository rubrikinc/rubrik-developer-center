# PrincipalEdge

Wrapper around the Principal object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                | Description                                       |
| ------ | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| cursor | String!                                                                                                             | String used to identify this edge.                |
| node   | [Principal](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/Principal/index.md)! | The actual Principal object wrapped by this edge. |
