# HierarchySnappableEdge

Wrapper around the HierarchySnappable object. This wrapper is used for pagination.

| Field  | Type                                                                                                                                     | Description                                                |
| ------ | ---------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| cursor | String!                                                                                                                                  | String used to identify this edge.                         |
| node   | [HierarchySnappable](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/interfaces/HierarchySnappable/index.md)! | The actual HierarchySnappable object wrapped by this edge. |
