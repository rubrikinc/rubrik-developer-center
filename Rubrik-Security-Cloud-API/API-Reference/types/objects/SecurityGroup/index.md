# SecurityGroup

SecurityGroup represents the security policy attached to a subnet on Azure.

## Fields

| Field | Type    | Description              |
| ----- | ------- | ------------------------ |
| id    | String! | The security group ID.   |
| name  | String! | The security group name. |
