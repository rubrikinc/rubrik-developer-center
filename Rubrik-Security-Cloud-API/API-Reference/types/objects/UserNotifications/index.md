# UserNotifications

An object representing product notifications.

## Fields

| Field       | Type                                                                                                      | Description                                              |
| ----------- | --------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| id          | String!                                                                                                   | The ID for the current user.                             |
| unreadCount | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md)! | The amount of unread notifications for the current user. |
