# MicrosoftGroupEdge

Wrapper around the MicrosoftGroup object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                             | Description                                            |
| ------ | -------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------ |
| cursor | String!                                                                                                                          | String used to identify this edge.                     |
| node   | [MicrosoftGroup](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/interfaces/MicrosoftGroup/index.md)! | The actual MicrosoftGroup object wrapped by this edge. |
