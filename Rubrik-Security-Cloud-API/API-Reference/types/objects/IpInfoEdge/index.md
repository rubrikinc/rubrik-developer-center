# IpInfoEdge

Wrapper around the IpInfo object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                          | Description                                    |
| ------ | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| cursor | String!                                                                                                       | String used to identify this edge.             |
| node   | [IpInfo](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/IpInfo/index.md)! | The actual IpInfo object wrapped by this edge. |
