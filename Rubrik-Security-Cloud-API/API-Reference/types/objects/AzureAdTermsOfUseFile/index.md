# AzureAdTermsOfUseFile

File of an Entra ID terms of use.

## Fields

| Field        | Type    | Description                                                                         |
| ------------ | ------- | ----------------------------------------------------------------------------------- |
| displayName  | String! | Display name of the Entra ID terms of use file.                                     |
| fileName     | String! | Name of the Entra ID terms of use file.                                             |
| languageCode | String! | Language code of the Entra ID terms of use file. It will be derived from ISO 639-1. |
