# M365Region

Represents Microsoft 365 region as described in the Microsoft Multi-Geo documentation.

## Fields

| Field | Type    | Description                                   |
| ----- | ------- | --------------------------------------------- |
| code  | String! | The three-letter code identifying the region. |
| name  | String! | The human-readable name of the region.        |
