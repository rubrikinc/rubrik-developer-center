# NcdVmImageUrl

NAS Cloud Direct virtual machine download URL information.

## Fields

| Field       | Type                                                                                                      | Description                                             |
| ----------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| downloadUrl | String!                                                                                                   | URL to download NAS Cloud Direct virtual machine image. |
| sha256      | String!                                                                                                   | Sha256 checksum value of the download image.            |
| size        | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md)! | Size of the NAS Cloud Direct virtual machine image.     |
