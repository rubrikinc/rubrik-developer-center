# RoleEdge

Wrapper around the Role object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                      | Description                                  |
| ------ | --------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| cursor | String!                                                                                                   | String used to identify this edge.           |
| node   | [Role](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/Role/index.md)! | The actual Role object wrapped by this edge. |
