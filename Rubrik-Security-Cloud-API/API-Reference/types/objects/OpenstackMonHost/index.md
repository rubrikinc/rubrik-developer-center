# OpenstackMonHost

Ceph monitor host configuration for OpenStack.

## Fields

| Field | Type    | Description                                                            |
| ----- | ------- | ---------------------------------------------------------------------- |
| ip    | String! | Required. Supported in v9.5+ The IP address of the Ceph monitor host.  |
| port  | Int!    | Required. Supported in v9.5+ The port number of the Ceph monitor host. |
