# GlobalSmbAuthSettings

Global SMB authentication settings for Cloud Direct site.

## Fields

| Field          | Type     | Description                             |
| -------------- | -------- | --------------------------------------- |
| hasCredentials | Boolean! | Whether SMB credentials are configured. |
| username       | String!  | SMB username (redacted for security).   |
