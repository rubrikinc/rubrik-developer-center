# FilterGroupConfig

FilterGroupConfig represents a group of filters with a logical operator.

## Fields

| Field           | Type                                                                                                                           | Description                                            |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------ |
| filtersList     | \[[PolicyFilter](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/PolicyFilter/index.md)!\]! | A list of filters in this group.                       |
| logicalOperator | [LogicalOperator](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/enums/LogicalOperator/index.md)!  | The logical operator applied to this group of filters. |
