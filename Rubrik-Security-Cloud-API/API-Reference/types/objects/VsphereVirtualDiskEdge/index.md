# VsphereVirtualDiskEdge

Wrapper around the VsphereVirtualDisk object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                                  | Description                                                |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| cursor | String!                                                                                                                               | String used to identify this edge.                         |
| node   | [VsphereVirtualDisk](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/VsphereVirtualDisk/index.md)! | The actual VsphereVirtualDisk object wrapped by this edge. |
