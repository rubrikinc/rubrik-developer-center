# O365AppEdge

Wrapper around the O365App object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                            | Description                                     |
| ------ | --------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| cursor | String!                                                                                                         | String used to identify this edge.              |
| node   | [O365App](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/O365App/index.md)! | The actual O365App object wrapped by this edge. |
