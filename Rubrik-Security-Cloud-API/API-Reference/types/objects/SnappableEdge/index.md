# SnappableEdge

Wrapper around the Snappable object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                | Description                                       |
| ------ | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| cursor | String!                                                                                                             | String used to identify this edge.                |
| node   | [Snappable](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/Snappable/index.md)! | The actual Snappable object wrapped by this edge. |
