# EulaState

A user's EULA acceptance state.

## Fields

| Field               | Type     | Description                                               |
| ------------------- | -------- | --------------------------------------------------------- |
| isAccepted          | Boolean! | Specifies whether a user has accepted the EULA.           |
| isPactsafeEnabled   | Boolean! | Specifies whether PactSafe EULA should be used.           |
| isPactsafeV2Enabled | Boolean! | Specifies whether PactSafe EULA version 2 should be used. |
