# RowEdge

Wrapper around the Row object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                    | Description                                 |
| ------ | ------------------------------------------------------------------------------------------------------- | ------------------------------------------- |
| cursor | String!                                                                                                 | String used to identify this edge.          |
| node   | [Row](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/Row/index.md)! | The actual Row object wrapped by this edge. |
