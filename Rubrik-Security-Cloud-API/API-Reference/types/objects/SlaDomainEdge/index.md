| ------ | ---------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| cursor | String!                                                                                                                | String used to identify this edge.                |
| node   | [SlaDomain](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/interfaces/SlaDomain/index.md)! | The actual SlaDomain object wrapped by this edge. |
