# NasSystemEdge

Wrapper around the NasSystem object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                | Description                                       |
| ------ | ------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| cursor | String!                                                                                                             | String used to identify this edge.                |
| node   | [NasSystem](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/NasSystem/index.md)! | The actual NasSystem object wrapped by this edge. |
