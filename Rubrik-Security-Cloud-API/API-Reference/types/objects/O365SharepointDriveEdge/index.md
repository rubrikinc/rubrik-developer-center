# O365SharepointDriveEdge

Wrapper around the O365SharepointDrive object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                                    | Description                                                 |
| ------ | --------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------- |
| cursor | String!                                                                                                                                 | String used to identify this edge.                          |
| node   | [O365SharepointDrive](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/O365SharePointDrive/index.md)! | The actual O365SharepointDrive object wrapped by this edge. |
