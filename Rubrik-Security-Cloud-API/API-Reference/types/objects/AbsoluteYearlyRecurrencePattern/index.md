# AbsoluteYearlyRecurrencePattern

## Fields

| Field      | Type    | Description                                |
| ---------- | ------- | ------------------------------------------ |
| dayOfMonth | Int!    | Which day of the month the event occurs.   |
| month      | String! | The month to which the recurrence applies. |
