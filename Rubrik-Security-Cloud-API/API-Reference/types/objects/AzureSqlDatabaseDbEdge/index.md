# AzureSqlDatabaseDbEdge

Wrapper around the AzureSqlDatabaseDb object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                                  | Description                                                |
| ------ | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| cursor | String!                                                                                                                               | String used to identify this edge.                         |
| node   | [AzureSqlDatabaseDb](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/AzureSqlDatabaseDb/index.md)! | The actual AzureSqlDatabaseDb object wrapped by this edge. |
