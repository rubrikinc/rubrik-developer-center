# MonthlyDaySpecSpecificDate

Specific date specification.

## Fields

| Field | Type                                                                                                                              | Description                                                                                                                                                                                                                                                                            |
| ----- | --------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| value | [SpecificDateSpec](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/SpecificDateSpec/index.md)! | Specifies which date of the month to take a snapshot on. Positive values denote an offset from the start of the month, while negative values denote an offset from the end of the month. For example, 2 denotes the second day of the month, and -1 denotes the last day of the month. |
