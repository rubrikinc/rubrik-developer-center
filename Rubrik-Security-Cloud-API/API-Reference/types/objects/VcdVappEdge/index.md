# VcdVappEdge

Wrapper around the VcdVapp object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                            | Description                                     |
| ------ | --------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| cursor | String!                                                                                                         | String used to identify this edge.              |
| node   | [VcdVapp](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/VcdVapp/index.md)! | The actual VcdVapp object wrapped by this edge. |
