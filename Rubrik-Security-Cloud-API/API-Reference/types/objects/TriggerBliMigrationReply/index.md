# TriggerBliMigrationReply

TriggerBLIMigrationResp is the response object for triggering blob level immutability migration for a list of RCV Azure locations.

## Fields

| Field   | Type     | Description                                                 |
| ------- | -------- | ----------------------------------------------------------- |
| success | Boolean! | Indicates whether the migration was triggered successfully. |
