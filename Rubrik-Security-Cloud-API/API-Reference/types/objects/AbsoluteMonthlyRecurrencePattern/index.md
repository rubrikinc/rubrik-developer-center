# AbsoluteMonthlyRecurrencePattern

An absolute monthly recurrence pattern (e.g. 21st of every 3 months).

## Fields

| Field      | Type | Description                                   |
| ---------- | ---- | --------------------------------------------- |
| dayOfMonth | Int! | Which day of the month the event occurs.      |
| interval   | Int! | The interval at which the recurrence applies. |
