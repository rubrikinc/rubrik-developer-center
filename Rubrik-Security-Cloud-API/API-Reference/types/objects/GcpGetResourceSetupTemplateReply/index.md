# GcpGetResourceSetupTemplateReply

GcpGetResourceSetupTemplateReply returns the Terraform template with the resource creation Terraform script.

## Fields

| Field    | Type    | Description                                              |
| -------- | ------- | -------------------------------------------------------- |
| template | String! | Resource setup template containing the Terraform script. |
