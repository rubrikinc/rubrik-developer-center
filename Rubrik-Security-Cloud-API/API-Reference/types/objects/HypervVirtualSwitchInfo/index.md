# HypervVirtualSwitchInfo

Information about a virtual switch on a Hyper-V host.

## Fields

| Field | Type    | Description                                            |
| ----- | ------- | ------------------------------------------------------ |
| id    | String! | Required. The unique identifier of the virtual switch. |
| name  | String! | Required. The name of the virtual switch.              |
