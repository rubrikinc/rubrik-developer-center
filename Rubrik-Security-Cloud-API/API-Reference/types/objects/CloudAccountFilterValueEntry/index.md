# CloudAccountFilterValueEntry

CloudAccountFilterValueEntry is a single filter entry with a user-facing display name and a filterable value.

## Fields

| Field       | Type    | Description                                         |
| ----------- | ------- | --------------------------------------------------- |
| displayName | String! | The user-facing display name for this filter entry. |
| value       | String! | The value used when applying this filter.           |
