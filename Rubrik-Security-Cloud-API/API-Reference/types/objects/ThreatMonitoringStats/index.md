# ThreatMonitoringStats

Aggregation stats for threat monitoring in the selected time range.

## Fields

| Field                | Type                                                                                                      | Description                                             |
| -------------------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------- |
| objectsWithMatches   | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md)! | Number of objects scanned with threat matches found.    |
| objectsWithNoMatches | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md)! | Number of objects scanned with no threat matches found. |
| totalObjectsScanned  | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md)! | Total number of objects scanned.                        |
