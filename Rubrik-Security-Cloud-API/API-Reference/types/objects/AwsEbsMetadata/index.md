# AwsEbsMetadata

AwsEbsMetadata represents the metadata for the AWS EBS volume.

## Fields

| Field           | Type    | Description                                                       |
| --------------- | ------- | ----------------------------------------------------------------- |
| attachedEc2Id   | String! | The ID of the EC2 instance to which the EBS volume is attached.   |
| attachedEc2Name | String! | The name of the EC2 instance to which the EBS volume is attached. |
