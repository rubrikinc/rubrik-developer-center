# GetCloudNativeTagRulesObjectTypeReply

GetCloudNativeTagRulesObjectTypeReply is the response to get the object type of the cloud native tag rule from its object id.

## Fields

| Field      | Type                                                                                                                              | Description                                                  |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------ |
| objectType | [ManagedObjectType](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/enums/ManagedObjectType/index.md)! | Object_type is the object type of the cloud native tag rule. |
