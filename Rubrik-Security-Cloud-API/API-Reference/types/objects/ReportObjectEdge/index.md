# ReportObjectEdge

Wrapper around the ReportObject object. This wrapper is used for pagination.

## Fields

| Field  | Type                                                                                                                      | Description                                          |
| ------ | ------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------- |
| cursor | String!                                                                                                                   | String used to identify this edge.                   |
| node   | [ReportObject](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/ReportObject/index.md)! | The actual ReportObject object wrapped by this edge. |
