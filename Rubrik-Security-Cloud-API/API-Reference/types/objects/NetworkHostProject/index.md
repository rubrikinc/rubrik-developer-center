# NetworkHostProject

Represents a GCP native network host project.

## Fields

| Field     | Type    | Description              |
| --------- | ------- | ------------------------ |
| name      | String! | Name of the GCP project. |
| nativeId  | String! | GCP native ID.           |
| projectId | String! | GCP project ID.          |
