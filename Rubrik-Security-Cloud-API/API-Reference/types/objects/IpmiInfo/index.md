# IpmiInfo

Cluster IPMI information.

## Fields

| Field       | Type     | Description                                          |
| ----------- | -------- | ---------------------------------------------------- |
| isAvailable | Boolean! | Indicates whether IPMI is available on this cluster. |
| usesHttps   | Boolean! | IPMI access via HTTPS.                               |
| usesIkvm    | Boolean! | IPMI access via iKVM.                                |
