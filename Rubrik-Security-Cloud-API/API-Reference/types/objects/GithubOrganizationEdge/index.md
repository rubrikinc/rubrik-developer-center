# GithubOrganizationEdge

| ------ | ------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------- |
| cursor | String!                                                                                                                               | String used to identify this edge.                         |
| node   | [GithubOrganization](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/GithubOrganization/index.md)! | The actual GithubOrganization object wrapped by this edge. |
