# VcdOrgEdge

## Fields

| Field  | Type                                                                                                          | Description                                    |
| ------ | ------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| cursor | String!                                                                                                       | String used to identify this edge.             |
| node   | [VcdOrg](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/VcdOrg/index.md)! | The actual VcdOrg object wrapped by this edge. |
