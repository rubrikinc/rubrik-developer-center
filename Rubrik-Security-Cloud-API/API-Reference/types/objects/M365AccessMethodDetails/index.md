# M365AccessMethodDetails

The details of how the principal can access an M365 item.

## Fields

| Field       | Type    | Description                                                   |
| ----------- | ------- | ------------------------------------------------------------- |
| itemName    | String! | The name of the M365 item where the access method is defined. |
| itemPath    | String! | The path of the M365 item where the access method is defined. |
| sharingLink | String! | The sharing link for the access method.                       |
