# CrossAccountClusterInfo

Cross-account Rubrik cluster information.

## Fields

| Field         | Type     | Description                                                                  |
| ------------- | -------- | ---------------------------------------------------------------------------- |
| isConnected   | Boolean! | Determines whether the Rubrik cluster is connected to RSC.                   |
| originAccount | String!  | Determines the RSC account from which the Rubrik cluster has been retrieved. |
