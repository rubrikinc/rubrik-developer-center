# RubrikSlaInfo

RubrikSlaInfo stores the SLA Domain information of the objects protected by Rubrik.

## Fields

| Field   | Type    | Description             |
| ------- | ------- | ----------------------- |
| slaId   | String! | ID of the SLA Domain.   |
| slaName | String! | Name of the SLA Domain. |
