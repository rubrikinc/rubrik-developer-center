# NewSsoGroupInput

Details of a new SSO Group that has not yet been authorized on Rubrik.

## Fields

| Field        | Type     | Description                                             |
| ------------ | -------- | ------------------------------------------------------- |
| authDomainId | String   | Authentication domain ID of the SSO group.              |
| isOrgAdmin   | Boolean! | Specifies whether the SSO group is an org admin or not. |
| name         | String!  | Name of the SSO group.                                  |
