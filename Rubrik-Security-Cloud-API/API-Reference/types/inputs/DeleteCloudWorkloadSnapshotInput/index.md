# DeleteCloudWorkloadSnapshotInput

The input for deleting Rubrik Security Cloud on-demand snapshot.

## Fields

| Field      | Type                                                                                                      | Description                                                               |
| ---------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| snapshotId | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | The forever UUID in Rubrik Security Cloud for the snapshot to be deleted. |
