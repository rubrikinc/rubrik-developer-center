# SetTotpConfigInput

Input for setting up TOTP configuration.

## Fields

| Field  | Type   | Description                                                        |
| ------ | ------ | ------------------------------------------------------------------ |
| otp    | String | TOTP one-time password.                                            |
| secret | String | Deprecated. This field is no longer supported and will be ignored. |
| userId | String | ID of the user to set up TOTP for.                                 |
