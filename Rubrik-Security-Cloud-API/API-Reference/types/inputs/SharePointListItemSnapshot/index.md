# SharePointListItemSnapshot

Represents the SharePoint list item snapshot to be restored.

## Fields

| Field       | Type                                                                                                      | Description                    |
| ----------- | --------------------------------------------------------------------------------------------------------- | ------------------------------ |
| snapshotId  | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | Id to snapshot to restore.     |
| snapshotNum | Int!                                                                                                      | Number of snapshot to restore. |
