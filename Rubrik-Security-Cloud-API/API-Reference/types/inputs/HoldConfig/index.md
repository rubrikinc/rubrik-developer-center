| Field             | Type                                                                                                                     | Description                                             |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------- |
| legalHoldMode     | [LegalHoldMode](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/enums/LegalHoldMode/index.md) | Specifies the legal hold mode for RSC native snapshots. |
