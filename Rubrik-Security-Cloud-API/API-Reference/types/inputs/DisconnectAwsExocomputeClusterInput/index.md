# DisconnectAwsExocomputeClusterInput

Input to disconnect a customer-managed cluster from RSC and mark it as terminated.

## Fields

| Field     | Type                                                                                                      | Description                                                                           |
| --------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------- |
| clusterId | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | Represents the ID of the customer-managed cluster that must be disconnected from RSC. |
