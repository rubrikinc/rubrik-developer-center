# FieldPreviewRequest

Preview request based on requested fields. Relevant only for structured files.

## Fields

| Field           | Type                                                                                                                                   | Description                                      |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| requestedFields | \[[FieldWithDataType](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/inputs/FieldWithDataType/index.md)!\] | Represents the list of fields to filter results. |
