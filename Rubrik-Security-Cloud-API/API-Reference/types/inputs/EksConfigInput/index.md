# EksConfigInput

The configuration of an Elastic Kubernetes Service (EKS) cluster.

## Fields

| Field          | Type    | Description                                                                                        |
| -------------- | ------- | -------------------------------------------------------------------------------------------------- |
| cloudAccountId | String! | Required. The cloud account for the Rubrik cluster to establish a connection with the EKS cluster. |
| eksClusterArn  | String! | Required. The Amazon Resource Name (ARN) for the EKS cluster.                                      |
