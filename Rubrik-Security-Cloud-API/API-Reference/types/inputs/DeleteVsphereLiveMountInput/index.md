# DeleteVsphereLiveMountInput

Input for deleting vSphere live mount.

## Fields

| Field | Type    | Description                                                                                                                |
| ----- | ------- | -------------------------------------------------------------------------------------------------------------------------- |
| force | Boolean | Force unmount to remove metadata when the datastore of the Live Mount virtual machine was moved off of the Rubrik cluster. |
| id    | String! | Required. ID of a Live Mount.                                                                                              |
