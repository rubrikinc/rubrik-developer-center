# AddAzureDevOpsCloudAccountInput

Contains parameters to create a new Azure DevOps cloud account configuration.

| ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |

Required when storage_type is BYOS; optional when storage_type is RCV (storage is auto-provisioned). Retrieve the ID by calling the allTargetMappings GraphQL query and using the id field of the desired TargetMapping.                                                                                                                                |
| backupRegion                  | String                                                                                                                                                        | Azure region for backup storage (e.g., "eastus", "westus2", "centralindia").
