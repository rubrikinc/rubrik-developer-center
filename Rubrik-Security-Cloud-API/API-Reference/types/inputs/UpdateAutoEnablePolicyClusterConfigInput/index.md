# UpdateAutoEnablePolicyClusterConfigInput

Input to update Rubrik cluster configuration.

## Fields

| Field     | Type    | Description                                                                                      |
| --------- | ------- | ------------------------------------------------------------------------------------------------ |
| clusterId | String  | Rubrik cluster ID.                                                                               |
| enabled   | Boolean | Specifies whether Auto-enabled Data Discovery Policies are enabled on the Rubrik cluster or not. |
