# CreateCrossAccountPairInput

OAuth authorization code input for cross-account pair creation.

## Fields

| Field     | Type    | Description                                                  |
| --------- | ------- | ------------------------------------------------------------ |
| code      | String! | Auth code generated by service provider RSC account.         |
| fqdn      | String! | Fully qualified domain name of service provider RSC account. |
| isRefresh | Boolean | Refresh cross-account pair.                                  |
| state     | String! | State of OAuth request.                                      |
