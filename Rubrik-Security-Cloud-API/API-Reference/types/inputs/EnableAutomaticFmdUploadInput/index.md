# EnableAutomaticFmdUploadInput

Enable/disable auto fmd upload on given cluster.

## Fields

| Field     | Type                                                                                                      | Description                           |
| --------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------- |
| clusterId | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | The uuid of the cluster.              |
| enabled   | Boolean!                                                                                                  | To enable or disable auto fmd upload. |
