# HostRbsNetworkLimitsInput

Network throttle limits for a host's Rubrik Backup Service.

## Fields

| Field           | Type                                                                                                     | Description                                                       |
| --------------- | -------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| throttlePercent | Int                                                                                                      | The percentage of available bandwidth that RBS is allowed to use. |
| throttleValue   | [Long](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/Long/index.md) | The value of available bandwidth that RBS is allowed to use.      |
