# CloudNativeTagCondition

CloudNativeTagCondition represents a condition with multiple tag key-value pairs for cloud-native tag rules.

## Fields

| Field    | Type                                                                                                                                      | Description                  |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------- |
| tagPairs | \[[CloudNativeTagPair](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/inputs/CloudNativeTagPair/index.md)!\]! | List of tag key-value pairs. |
