# UpdateClusterDefaultAddressInput

Object containing the new address of the Rubrik cluster.

## Fields

| Field       | Type                                                                                                      | Description                                       |
| ----------- | --------------------------------------------------------------------------------------------------------- | ------------------------------------------------- |
| address     | String                                                                                                    | The new default IP address of the Rubrik cluster. |
| clusterUuid | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | UUID of the cluster.                              |
| port        | Int                                                                                                       | The new default port of the Rubrik cluster.       |
