# DisableSupportUserAccessInput

Input for disabling a Rubrik Support representative's access to the customer account.

## Fields

| Field              | Type   | Description                                                                                                  |
| ------------------ | ------ | ------------------------------------------------------------------------------------------------------------ |
| id                 | Int    | Support user access ID.                                                                                      |
| impersonatedUserId | String | User ID of the customer on whose behalf the Rubrik Support representative is accessing the customer account. |
