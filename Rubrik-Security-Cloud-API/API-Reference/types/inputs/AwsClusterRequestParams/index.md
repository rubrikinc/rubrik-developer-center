# AwsClusterRequestParams

AWS specific Exocompute cluster customizations.

## Fields

| Field           | Type   | Description                                                                                                                                |
| --------------- | ------ | ------------------------------------------------------------------------------------------------------------------------------------------ |
| nodeKeypairName | String | SSH key pair name to be used for launching EC2 worker nodes in Exocompute cluster. This key pair can be used to SSH into the worker nodes. |
