# ClusterUuidWithDbIdInput

Contains the cluster UUID and Oracle database ID.

| Field       | Type                                                                                                      | Description                                                                     |

| clusterUuid | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | Required. Specifies UUID used to identify the cluster that the request goes to. |
| dbId        | String!                                                                                                   | ID of the Oracle database.                                                      |
