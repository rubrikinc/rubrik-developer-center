# VsphereExcludeVmDisksInput

Set disks to be included/excluded in snapshot.

## Fields

| Field                | Type                                                                                                      | Description                                  |
| -------------------- | --------------------------------------------------------------------------------------------------------- | -------------------------------------------- |
| excludeFromSnapshots | Boolean                                                                                                   | Virtual disk will be excluded from snapshot. |
| virtualDiskFid       | [UUID](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/UUID/index.md)! | Fid of virtual disk.                         |
