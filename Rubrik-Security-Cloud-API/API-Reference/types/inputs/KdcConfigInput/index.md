# KdcConfigInput

Input KDC configuration for Kerberos authentication.

## Fields

| Field | Type    | Description                              |
| ----- | ------- | ---------------------------------------- |
| kdc1  | String! | Primary KDC server address.              |
| kdc2  | String  | Secondary KDC server address (optional). |
| realm | String! | Kerberos realm name.                     |
