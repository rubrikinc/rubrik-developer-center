# TimeSpanFilter

Defines a time-bounded filter with optional from and until times.

## Fields

| Field     | Type                                                                                                             | Description                               |
| --------- | ---------------------------------------------------------------------------------------------------------------- | ----------------------------------------- |
| fromTime  | [DateTime](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/DateTime/index.md) | Represents the lower bound of the filter. |
| untilTime | [DateTime](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/scalars/DateTime/index.md) | Represents the upper bound of the filter. |
