# ActivityAuditorAttributeChangeFilter

Filter for attribute changes.

## Fields

| Field          | Type      | Description                               |
| -------------- | --------- | ----------------------------------------- |
| attributeNames | [String!] | The names of the attributes that changed. |
| newValueSubset | String    | A subset of the attribute's new value.    |
| oldValueSubset | String    | A subset of the attribute's old value.    |
