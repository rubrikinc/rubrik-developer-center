# UserFieldEnum

User fields for sorting.

## Values

| Value     | Description                                                 |
| --------- | ----------------------------------------------------------- |
| Domain    | The user's organizational domain or network segment.        |
| Email     | The primary email address associated with the user account. |
| LastLogin | The timestamp of the user's most recent login.              |
| Status    | The current status of the user account.                     |
