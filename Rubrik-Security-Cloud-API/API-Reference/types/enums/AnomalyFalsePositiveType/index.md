| LOG_ROTATION                | Log deletion/rotation.           |
| NFA_SCHEDULED_MAINTENANCE   | Scheduled maintenance.           |
| NFA_UNSCHEDULED_MAINTENANCE | Unscheduled maintenance.         |
| OS_UPDATE                   | OS update.                       |
| OTHER                       | Another false positive type.     |
