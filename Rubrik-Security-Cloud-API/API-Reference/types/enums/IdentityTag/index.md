# IdentityTag

Identity tag specifies the tags for the identity.

## Values

| Value                    | Description                               |
| ------------------------ | ----------------------------------------- |
| AT_RISK                  | Identities with open violations.          |
| IDENTITY_TAG_UNSPECIFIED | Unspecified identity tag.                 |
| PRIVILEGED               | Identities with privileged access.        |
| SENSITIVE                | Identities with access to sensitive data. |
