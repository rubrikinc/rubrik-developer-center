# SensitivityLevel

Represents the sensitivity level of a resource.

## Values

| Value               | Description          |
| ------------------- | -------------------- |
| HIGH_SENSITIVITY    | High sensitivity.    |
| LOW_SENSITIVITY     | Low sensitivity.     |
| MEDIUM_SENSITIVITY  | Medium sensitivity.  |
| NO_SENSITIVITY      | No sensitivity.      |
| UNKNOWN_SENSITIVITY | Unknown sensitivity. |
