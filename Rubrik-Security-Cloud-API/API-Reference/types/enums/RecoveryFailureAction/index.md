# RecoveryFailureAction

It specifies the action to take when the recovery fails.

## Values

| Value       | Description                                                           |
| ----------- | --------------------------------------------------------------------- |
| CLEANUP     | Specifies the recovery failure action as cleanup when failure occurs. |
| IGNORE      | Specifies the recovery failure action as ignore when failure occurs.  |
| PAUSE       | Specifies the recovery failure action as pause when failure occurs.   |
| UNSPECIFIED | Recovery failure action unspecified.                                  |
