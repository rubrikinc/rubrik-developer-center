# NfAnomalyResultGroupBy

Group non-filesystem anomaly results by field.

## Values

| Value        | Description                                 |
| ------------ | ------------------------------------------- |
| CLUSTER_UUID | The unique ID of the cluster.               |
| DAY          | Group by day.                               |
| HOUR         | Group by hour.                              |
| IS_ANOMALY   | Specifies whether the result is an anomaly. |
| MONTH        | Group by month.                             |
| WEEK         | Group by week.                              |
| YEAR         | Group by year.                              |
