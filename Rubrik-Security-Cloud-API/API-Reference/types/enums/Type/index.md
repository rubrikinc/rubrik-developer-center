# Type

Directory object property data type.

## Values

| Value             | Description                                                                       |
| ----------------- | --------------------------------------------------------------------------------- |
| BOOLEAN           | Boolean data type.                                                                |
| INTEGER           | Integer data type.                                                                |
| ISO_8601_DATETIME | Date time data type. Expected format needs to be yyyy-mm-ddThhss('.'s+)?(zzzzzz)? |
| STRING            | String data type.                                                                 |
