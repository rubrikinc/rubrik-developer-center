# KosmosTopologyReplicaRole

Role of a replica within a Kosmos HA topology.

## Values

| Value       | Description                                                 |
| ----------- | ----------------------------------------------------------- |
| PRIMARY     | Replica is the primary read-write node for the HA cluster.  |
| SECONDARY   | Replica is a secondary node (replication target / standby). |
| UNSPECIFIED | Role could not be determined or is unrecognized.            |
