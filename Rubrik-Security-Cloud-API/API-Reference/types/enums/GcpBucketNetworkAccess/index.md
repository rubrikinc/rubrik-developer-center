# GcpBucketNetworkAccess

GcpBucketNetworkAccess defines the network access type for a GCP bucket.

## Values

| Value                             | Description                                       |
| --------------------------------- | ------------------------------------------------- |
| GCP_BUCKET_PRIVATE                | Private access (restricted to specific networks). |
| GCP_BUCKET_PUBLIC                 | Public access enabled (all networks).             |
| UNKNOWN_GCP_BUCKET_NETWORK_ACCESS | Network access setting is unknown.                |
