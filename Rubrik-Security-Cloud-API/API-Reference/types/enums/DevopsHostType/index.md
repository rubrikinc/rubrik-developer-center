# DevopsHostType

Type of exocompute host used for DevOps workload protection.

## Values

| Value                 | Description                 |
| --------------------- | --------------------------- |
| CUSTOMER_HOST         | Customer hosted exocompute. |
| HOST_TYPE_UNSPECIFIED | Unspecified host type.      |
| RUBRIK_HOST           | Rubrik hosted exocompute.   |
