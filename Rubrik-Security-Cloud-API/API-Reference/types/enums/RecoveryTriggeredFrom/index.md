# RecoveryTriggeredFrom

TriggeredFrom specifies how the recovery was triggered.

## Values

| Value                 | Description                                        |
| --------------------- | -------------------------------------------------- |
| ORCHESTRATED_RECOVERY | Recovery was triggered from Orchestrated Recovery. |
| RANSOMWARE_MONITORING | Recovery was triggered from Ransomware Monitoring. |
| RECOVERY_SCHEDULER    | Recovery was triggered from Recovery Scheduler.    |
| TRIGGER_UNSPECIFIED   | Unspecified value (Should not be used).            |
