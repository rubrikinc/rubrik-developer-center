# SigninLogRiskLevel

Risk level of a sign-in event.

## Values

| Value                        | Description           |
| ---------------------------- | --------------------- |
| SIGNIN_LOG_RISK_LEVEL_HIGH   | High risk detected.   |
| SIGNIN_LOG_RISK_LEVEL_LOW    | Low risk detected.    |
| SIGNIN_LOG_RISK_LEVEL_MEDIUM | Medium risk detected. |
| SIGNIN_LOG_RISK_LEVEL_NONE   | No risk detected.     |
