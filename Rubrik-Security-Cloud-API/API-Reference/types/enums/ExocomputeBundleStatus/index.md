# ExocomputeBundleStatus

ExocomputeBundleStatus represents the status of the Exocompute bundle version.

## Values

| Value                         | Description                                         |
| ----------------------------- | --------------------------------------------------- |
| BUNDLE_STATUS_LATEST          | Bundle version is the latest.                       |
| BUNDLE_STATUS_OUTDATED        | Bundle version is outdated and needs to be updated. |
| BUNDLE_STATUS_UNKNOWN         | Unknown status.                                     |
| BUNDLE_STATUS_UPDATE_REQUIRED | Bundle version update is required.                  |
