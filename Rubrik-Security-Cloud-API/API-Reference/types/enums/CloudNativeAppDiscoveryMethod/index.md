# CloudNativeAppDiscoveryMethod

CloudNativeAppDiscoveryMethod mirrors cloudnativeservice.DiscoveryMethod. Defined here to avoid an import cycle (cloudnativeservice already imports authzservice).

## Values

| Value                                         | Description                   |
| --------------------------------------------- | ----------------------------- |
| CLOUD_NATIVE_APP_AUTO_DISCOVERY               | Auto discovery.               |
| CLOUD_NATIVE_APP_DISCOVERY_METHOD_UNSPECIFIED | Unspecified discovery method. |
| CLOUD_NATIVE_APP_TAG_BASED                    | Tag based discovery.          |
