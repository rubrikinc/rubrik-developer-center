# ActivityOperation

The operation of the activity.

## Values

| Value                          | Description            |
| ------------------------------ | ---------------------- |
| ACTIVITY_OPERATION_ADD         | Add operation.         |
| ACTIVITY_OPERATION_CHANGE      | Change operation.      |
| ACTIVITY_OPERATION_REMOVE      | Remove operation.      |
| ACTIVITY_OPERATION_UNSPECIFIED | Unspecified operation. |
