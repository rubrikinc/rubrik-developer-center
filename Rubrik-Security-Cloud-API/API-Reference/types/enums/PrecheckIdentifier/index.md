# PrecheckIdentifier

Identifier for the precheck.

## Values

| Value                | Description                                                       |
| -------------------- | ----------------------------------------------------------------- |
| PRECHECK_MIGRATION   | RSC migration precheck.                                           |
| PRECHECK_OTHER       | CDM precheck.                                                     |
| PRECHECK_UNSPECIFIED | This is not used. Added due to backward compatibility guidelines. |
