# DefenderAlertSeverity

Defender supports 4 severity levels (no critical). UI dropdown values: INFORMATIONAL, LOW, MEDIUM, HIGH.

## Values

| Value                           | Description             |
| ------------------------------- | ----------------------- |
| DEFENDER_SEVERITY_HIGH          | High severity.          |
| DEFENDER_SEVERITY_INFORMATIONAL | Informational severity. |
| DEFENDER_SEVERITY_LOW           | Low severity.           |
| DEFENDER_SEVERITY_MEDIUM        | Medium severity.        |
