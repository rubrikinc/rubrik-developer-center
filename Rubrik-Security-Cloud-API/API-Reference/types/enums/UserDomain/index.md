# UserDomain

Domain type of a user.

## Values

| Value   | Description                        |
| ------- | ---------------------------------- |
| CLIENT  | Client user domain.                |
| LDAP    | LDAP user domain.                  |
| LOCAL   | Local user domain.                 |
| PAT     | Personal access token user domain. |
| SSO     | SSO user domain.                   |
| SUPPORT | Support user domain.               |
