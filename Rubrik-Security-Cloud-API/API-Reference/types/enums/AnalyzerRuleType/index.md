# AnalyzerRuleType

Represents the type of data you need to analyze using this data type.

## Values

| Value        | Description                               |
| ------------ | ----------------------------------------- |
| STRUCTURED   | Use data type only for structured data.   |
| UNSTRUCTURED | Use data type only for unstructured data. |
