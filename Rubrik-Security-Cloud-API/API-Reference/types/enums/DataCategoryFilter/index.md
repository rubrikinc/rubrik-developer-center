# DataCategoryFilter

Controls which data categories are returned.

## Values

| Value                    | Description                                         |
| ------------------------ | --------------------------------------------------- |
| ACTIVE_DATA_CATEGORIES   | Return only active data categories (default).       |
| ALL_DATA_CATEGORIES      | Return all data categories excluding deleted ones.  |
| INACTIVE_DATA_CATEGORIES | Return only inactive (non-deleted) data categories. |
