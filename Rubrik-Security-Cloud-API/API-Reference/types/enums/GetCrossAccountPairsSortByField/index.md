# GetCrossAccountPairsSortByField

Field to sort by for cross-account pairs.

## Values

| Value                     | Description                  |
| ------------------------- | ---------------------------- |
| NAME                      | Cross-account pair name.     |
| ORG_NAME                  | Cross-account pair org name. |
| SORT_BY_FIELD_UNSPECIFIED | Unspecified sort by field.   |
