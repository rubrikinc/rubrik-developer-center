# RecoveryRangeStatus

Status of a recovery range.

## Values

| Value                                 | Description                                              |
| ------------------------------------- | -------------------------------------------------------- |
| NOT_RECOVERABLE_DUE_TO_BACKUP_FAILURE | Recovery range is not recoverable due to backup failure. |
| NOT_RECOVERABLE_DUE_TO_OTHER_REASON   | Recovery range is not recoverable due to other reason.   |
| NOT_RECOVERABLE_DUE_TO_PROCESSING     | Recovery range is not recoverable due to processing.     |
| NOT_RECOVERABLE_DUE_TO_QUARANTINE     | Recovery range is not recoverable due to quarantine.     |
| RECOVERABLE                           | Recovery range is recoverable.                           |
| UNSPECIFIED                           | Unspecified status.                                      |
