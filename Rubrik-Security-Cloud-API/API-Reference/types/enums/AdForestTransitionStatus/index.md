# AdForestTransitionStatus

Active Directory forest transition status.

## Values

| Value               | Description                                                                            |
| ------------------- | -------------------------------------------------------------------------------------- |
| CLUSTER_UNSUPPORTED | Active Directory forest transition is invalid because some clusters are not supported. |
| INTERNAL_ERROR      | Active Directory forest transition is invalid because of some internal error.          |
| OK                  | Active Directory forest transition is valid.                                           |
