# NfsSubType

Subtypes for NFS archival locations, representing the NFS vendor.

## Values

| Value                | Description                                  |
| -------------------- | -------------------------------------------- |
| NFS_DELL_DATA_DOMAIN | Dell Data Domain NFS subtype.                |
| NFS_ISILON           | Isilon NFS subtype.                          |
| NFS_TYPE_UNSPECIFIED | Unspecified or default NFS subtype (Others). |
| NFS_VAST_DATA        | VAST Data NFS subtype.                       |
