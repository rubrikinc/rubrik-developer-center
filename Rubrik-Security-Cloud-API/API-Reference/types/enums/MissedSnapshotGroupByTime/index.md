# MissedSnapshotGroupByTime

Group missed snapshots by time.

## Values

| Value   | Description       |
| ------- | ----------------- |
| DAY     | Group by day.     |
| HOUR    | Group by hour.    |
| MONTH   | Group by month.   |
| QUARTER | Group by quarter. |
| WEEK    | Group by week.    |
| YEAR    | Group by year.    |
