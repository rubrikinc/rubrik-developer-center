# FusionComputeVmStatus

Status of the FusionCompute live mount.

## Values

| Value                                | Description                |
| ------------------------------------ | -------------------------- |
| FUSION_COMPUTE_VM_STATUS_UNSPECIFIED | Unspecified status.        |
| MOUNTING                             | Live mount is mounting.    |
| POWERED_OFF                          | Live mount is powered off. |
| POWERED_ON                           | Live mount is powered on.  |
| UNMOUNTING                           | Live mount is unmounting.  |
