# AzureAdPimAssignmentType

How a PIM active assignment was created.

## Values

| Value                           | Description                                                   |
| ------------------------------- | ------------------------------------------------------------- |
| PIM_ASSIGNMENT_TYPE_ACTIVATED   | Temporary assignment activated from an eligibility.           |
| PIM_ASSIGNMENT_TYPE_ASSIGNED    | Permanent or scheduled admin-assigned active assignment.      |
| PIM_ASSIGNMENT_TYPE_UNSPECIFIED | Indicates the assignment type was not set or is unrecognized. |
