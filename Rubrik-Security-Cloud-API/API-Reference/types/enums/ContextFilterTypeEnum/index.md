# ContextFilterTypeEnum

Context filter enum for SLA.

## Values

| Value                    | Description                                                                           |
| ------------------------ | ------------------------------------------------------------------------------------- |
| APPFLOWS_FAILOVER_TO_AWS | Context filter for SLAs supporting orchestrated application recovery failover to AWS. |
| APPFLOWS_FAILOVER_TO_CDM | Context filter for SLAs supporting orchestrated application recovery failover to CDM. |
| DEFAULT                  | Do not change the DEFAULT_TYPE numbering.                                             |
