# CertificateUsageLocation

The location where a certificate is used.

## Values

| Value                                  | Description                                       |
| -------------------------------------- | ------------------------------------------------- |
| CDM                                    | The certificate is used on Rubrik CDM clusters.   |
| CERTIFICATE_USAGE_LOCATION_UNSPECIFIED | The location is unspecified.                      |
| RSC                                    | The certificate is used on Rubrik Security Cloud. |
