# DiscoveryReportGroupBy

Group-by field for the report.

## Values

| Value             | Description                                           |
| ----------------- | ----------------------------------------------------- |
| POLICY_VIOLATIONS | Group by policy and count violations for each policy. |
| STATUS_POLICY     | Group by policy and count statuses for each policy.   |
| TIME_ISSUES       | Group by time bucket and count issues per bucket.     |
| TIME_VIOLATIONS   | Group by time bucket and count violations per bucket. |
