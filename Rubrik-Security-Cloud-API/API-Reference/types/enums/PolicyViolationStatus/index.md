# PolicyViolationStatus

Represents the possible statuses of a policy violation.

## Values

| Value                               | Description                                      |
| ----------------------------------- | ------------------------------------------------ |
| POLICY_VIOLATION_STATUS_CLOSED      | The violation has been closed.                   |
| POLICY_VIOLATION_STATUS_DISMISSED   | The violation has been dismissed by the user.    |
| POLICY_VIOLATION_STATUS_IN_PROGRESS | The remediation of the violation is in progress. |
| POLICY_VIOLATION_STATUS_OPEN        | The violation is open.                           |
| POLICY_VIOLATION_STATUS_REMEDIATED  | The violation has been remediated.               |
| POLICY_VIOLATION_STATUS_UNSPECIFIED | Unspecified violation status.                    |
