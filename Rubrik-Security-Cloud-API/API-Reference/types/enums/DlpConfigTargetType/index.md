# DlpConfigTargetType

Specifies the type of the target that files will be recovered to. Note, this also specifies which configuration contains valid data.

## Values

| Value                   | Description                                                             |
| ----------------------- | ----------------------------------------------------------------------- |
| GENERIC_NAS             | Generic NAS share target.                                               |
| TARGET_TYPE_UNSPECIFIED | The TARGET_TYPE_UNSPECIFIED type represents an unspecified target type. |
| VMWARE_VM               | VMware virtual machine target.                                          |
