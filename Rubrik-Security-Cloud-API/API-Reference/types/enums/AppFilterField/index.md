# AppFilterField

Parameter to filter Azure apps on.

## Values

| Value        | Description                  |
| ------------ | ---------------------------- |
| APP_ID       | Filter on app ID.            |
| APP_TYPE     | Filter on app type.          |
| ORG_ID       | Filter by Org ID.            |
| SUBSCRIPTION | Filter on subscription name. |
| TENANT_ID    | Filter on tenant ID.         |
