# MongoNodeType

Represents the type of the MongoDB node.

## Values

| Value                 | Description                       |
| --------------------- | --------------------------------- |
| CONFIG_SERVER         | Config server node.               |
| MONGOS                | Mongos router node.               |
| NODE_TYPE_UNSPECIFIED | Unknown or unspecified node type. |
| REPLICA_SET           | Replica set member node.          |
