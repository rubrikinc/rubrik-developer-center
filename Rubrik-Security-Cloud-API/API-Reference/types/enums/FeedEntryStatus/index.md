# FeedEntryStatus

Status of the feed entry.

## Values

| Value              | Description                       |
| ------------------ | --------------------------------- |
| ACTIVE             | Active feed entry.                |
| DELETED            | Deleted feed entry.               |
| DISABLED_BY_RUBRIK | Feed entry deactivated by Rubrik. |
| DISABLED_BY_USER   | Feed entry deactivated by user.   |
