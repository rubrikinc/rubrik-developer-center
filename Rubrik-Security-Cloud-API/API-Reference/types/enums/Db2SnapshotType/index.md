# Db2SnapshotType

Enum for db2 snapshot type.

## Values

| Value        | Description                                                                 |
| ------------ | --------------------------------------------------------------------------- |
| DIFFERENTIAL | Db2 Snapshot since last successful full snapshot.                           |
| FULL         | Db2 Full Snapshot.                                                          |
| INCREMENTAL  | Db2 Snapshot since any (full/differential/incremental) successful snapshot. |
