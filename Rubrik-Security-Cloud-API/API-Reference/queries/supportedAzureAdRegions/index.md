# supportedAzureAdRegions

Lists all the supported Azure AD regions.

## Returns

[SupportedAzureAdRegions](https://developer.rubrik.com/Rubrik-Security-Cloud-API/API-Reference/types/objects/SupportedAzureAdRegions/index.md)!

## Sample

```graphql
query {
  supportedAzureAdRegions {
    regions
  }
}
```

```json
{}
```

```json
{
  "data": {
    "supportedAzureAdRegions": {
      "regions": [
        "AUSTRALIAEAST"
      ]
    }
  }
}
```
