## [Rubrik Clusters](https://developer.rubrik.com/Rubrik-Security-Cloud-API/Data-Protection/Infrastructure/Clusters/index.md)

Rubrik Clusters are the physical or virtual machines that run the Rubrik software. Rubrik Clusters can be physical or virtual, and can be located on-premises or in the cloud.

## [Rubrik Cloud Vault](https://developer.rubrik.com/Rubrik-Security-Cloud-API/Data-Protection/Infrastructure/Rubrik-Cloud-Vault/index.md)

Rubrik Cloud Vault is a service that provides a secure, scalable, and cost-effective way to archive Rubrik backups.
